# Task 3: 实现截图捕获管线 (周期截图 + pHash 去重 + VLM 理解)

> **Status**: completed | **Priority**: P0 | **Complexity**: 5

## Context
**What**: 扩展 electron-screen-capture，实现周期截图→pHash 去重→AI SDK 6 VLM 内容提取→存入 LanceDB
**Why**: 上下文感知引擎的核心数据来源 (F3)，角色需要"看到"用户在做什么
**Constraints**:
- 截图频率可配置 (默认 60s)
- pHash 去重: 相似度 >95% 跳过 VLM 调用 (节省 70-80% VLM 成本)
- VLM 用 AI SDK 6 的 generateText with image

## Implementation
**Target Files**:
- `packages/context-engine/src/capture/screenshot.ts` (create) — 周期截图调度 + pHash 计算
- `packages/context-engine/src/processing/screenshot-processor.ts` (create) — VLM 理解 + 实体提取
- `packages/context-engine/src/capture/phash.ts` (create) — 感知哈希实现 (用 sharp)
- `packages/electron-screen-capture/src/main/index.ts` (modify: 导出周期截图 API)

**Pattern**: 参考已有 `packages/electron-screen-capture` 的 desktopCapturer 用法
**Tech Notes**: sharp 做图片缩放+pHash；AI SDK 6 `generateText({ model, messages: [{ role: 'user', content: [{ type: 'image', image: buffer }] }] })`
