# Task 9: 创建 mcp-hub 包 + McpRegistry 多服务器管理

> **Status**: completed | **Priority**: P0 | **Complexity**: 5

## Context
**What**: 创建 MCP Hub 包，实现 McpRegistry (多服务器注册表+持久化) + McpClientManager (AI SDK MCP Client 管理) + TransportFactory (stdio/SSE/HTTP)
**Why**: 无限扩展能力的"连接层" (F2, ADR-004)
**Constraints**:
- 配置持久化到 SQLite (document-store)
- 三种传输: stdio (本地进程) / SSE / Streamable HTTP
- 工具聚合: 所有 Server 的 tools 合并传给 LLM
- 懒加载: 不活跃 Server 不启动

## Implementation
**Target Files**:
- `packages/mcp-hub/package.json` (create)
- `packages/mcp-hub/src/mcp-registry.ts` (create) — 多服务器配置 CRUD + 持久化
- `packages/mcp-hub/src/mcp-client-manager.ts` (create) — AI SDK createMCPClient 管理
- `packages/mcp-hub/src/transport-factory.ts` (create) — stdio/SSE/HTTP 传输创建
- `packages/mcp-hub/src/index.ts` (create)

**Pattern**: 参考 `packages/stage-ui/src/tools/mcp.ts` 的现有 MCP 集成
**Tech Notes**: `@ai-sdk/mcp` 的 createMCPClient; stdio 用 child_process.spawn; 健康检查用 ping/pong
