# Task 16: 创建 channels-extra 包 + WhatsApp + Slack 频道

> **Status**: completed | **Priority**: P1 | **Complexity**: 5

## Context
**What**: 创建 channels-extra 包，定义统一 Channel 接口，实现 WhatsApp (baileys) 和 Slack (@slack/bolt) 频道
**Why**: 多频道消息触达——用户不在桌面时仍可交互 (F10, F11, Scenario 5)
**Constraints**:
- 统一 Channel 接口: connect/disconnect/sendMessage/onMessage
- WhatsApp 用 @whiskeysockets/baileys (注意封号风险，需限流)
- Slack 用 @slack/bolt (官方 SDK)
- 频道配置持久化到 SQLite

## Implementation
**Target Files**:
- `packages/channels-extra/package.json` (create)
- `packages/channels-extra/src/base-channel.ts` (create) — 统一 Channel 接口

**Pattern**: 参考 `services/discord-bot` 和 `services/telegram-bot` 的 bot 实现模式
